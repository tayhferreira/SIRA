Sistema Inteligente de Redistribuição de Alimentos (SIRA)
Descrição do Projeto

O SIRA é uma solução tecnológica desenvolvida para o Projeto Integrador:
Desenvolvimento de Sistemas Orientado a Objetos. 

O objetivo central é gerenciar a redistribuição eficiente de excedentes alimentares, 
intermediando o contato entre estabelecimentos comerciais e instituições sociais.
A aplicação foca na redução do desperdício por meio de um sistema de disponibilidade,
geolocalização e rastreabilidade.

Aspectos Técnicos

O desenvolvimento seguiu as diretrizes da Engenharia de Software e os pilares da Programação Orientada a Objetos (POO).
•	Linguagem: C# (.NET 9.0).
•	Persistência: Proposta baseada em Entity Framework Core e classes POCO.
•	Arquitetura: Modelagem focada no domínio com alto nível de encapsulamento e coesão.

Estrutura de Classes

As entidades principais do sistema foram projetadas para garantir a integridade dos dados e a escalabilidade da solução:

•	Usuario: Classe abstrata para gestão de identidades e herança.
•	Estabelecimento: Gestão de doadores e postagem de lotes.
•	Instituicao: Gestão de receptores e solicitações de reserva.
•	Alimento: Controle de validade, quantidade e status de itens.
•	Transacao: Controle logístico e rastreabilidade da entrega.

Instruções de Execução

Para executar o protótipo funcional:

1.	Verifique se o SDK do .NET 9.0 está instalado no ambiente.
2.	Clone o repositório para o diretório local.
3.	Abra a solução no JetBrains Rider ou Visual Studio.
4.	Execute o projeto SIRA.Domain.
